# Changelog

## 2025-11-12

- Добавлена полноценная документация (MkDocs + PDF) для проекта oil-simulator-py-impes.
- Описаны физические и математические модели, архитектура, производительность.
- Интегрирована система экспорта PDF через Playwright.
- Зафиксированы параметры миллионного AMG-кейса и ограничения по памяти.

## 2025-11-10

- Завершена оптимизация `_csr_spmm` (блочная обработка без `coalesce`).
- Решена проблема `CUDA out of memory` при построении RAP на 1e6 ячеек.
- Обновлены конфиги для стресс-теста `mega_3phase_million`.

## 2025-11-05

- Реализовано кэширование `ClassicalAMG` и метод `update_matrix`.
- Переписан `build_prolongation_rs_full` на PyTorch без Python-циклов.
- Введена диагностика качества (`const_rel`, `grad_rel`) во время построения.

## 2025-10-09

- Расширены тесты (`test_impes_3phase`, `test_component_balance_rv`).
- Добавлены крупные конфиги (`impes_large_multiwell_3d.json`, `large_3d.json`).

## 2025-09-20

- Начальная интеграция AMG в IMPES-симулятор (Ruge–Stuben, Jacobi smoother).
- Настроена генерация отчётов и визуализация результатов.
