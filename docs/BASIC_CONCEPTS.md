# Основные понятия

## Ячейка (Cell)

Минимальный объём сетки, для которого хранятся:

- давление `p`;
- насыщенности `S_w`, `S_g` (нефтяная определяется как остаток);
- пористость $\phi$, проницаемость $k$;
- координаты центра `(x, y, z)`.

## Трансмиссивити (Transmissibility)

Коэффициент, характеризующий поток между двумя соседними ячейками. Зависит от площади грани, расстояния между центрами, проницаемости и вязкости.

## Суммарная подвижность (Total Mobility)

$$
\lambda_t = \sum_{\alpha} \frac{k_{r,\alpha}(S)}{\mu_\alpha(p)}
$$

Определяет коэффициенты в уравнении давления.

## Сильные связи (Strong Connections)

Соседние узлы матрицы, влияющие на выбор C-точек. Если $| -a_{ij} |$ велик по сравнению с соседями, связь считается «сильной».

## C-точки и F-точки

- **C-точки** — узлы, которые остаются в грубом уровне.
- **F-точки** — узлы, значения которых интерполируются через C-точки.

## Near-nullspace

Векторы, которые матрица почти не изменяет (константы, линейные градиенты). AMG должен воспроизводить их без искажений.

## V-цикл

Одна итерация AMG: предсглаживание → рестрикция остатка → решение на грубом уровне → пролонгация коррекции → постсглаживание.

## Кэш AMG

Сохранённая структура иерархии (`P`, `R`, `A_c`) для повторного использования. Значительно ускоряет серийные расчёты.

## Расчёт шага

IMPES выполняет:

1. Решение давления (`A x = b`) с AMG.
2. Обновление насыщенностей (явным методом).
3. Пересчёт свойств флюидов (PVT, relPerm).
4. Корректировку скважин (дебиты, накопительные показатели).
