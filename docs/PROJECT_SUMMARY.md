# Сводка проекта

## Назначение

Oil Simulator — это GPU-ускоренный исследовательский код для моделирования многокомпонентных потоков в пористых средах с упором на качественный AMG-предобуславливатель дла решения уравнений давления.

## Ключевые особенности

- Трёхфазная IMPES-схема (implicit pressure, explicit saturation).
- Собственная реализация Classical AMG на PyTorch с поддержкой CUDA.
- Гибкая конфигурация сценариев (JSON).
- Набор автоматических тестов для валидации физики и численной устойчивости.

## Архитектура

- `simulator/`: основная физика (резервуар, флюид, скважины, временной интегратор).
- `solver/`: AMG и интерфейсы линейных решателей.
- `plotting/`, `output/`: визуализация и сохранение результатов.
- `configs/`: шаблонные задачи (2D, 3D, стресс-тесты).

## Технологии

- Python 3.10+, PyTorch (sparse CSR), CUDA 11.8+
- MkDocs Material для документации и Playwright для PDF
- PyTest для тестов

## Текущий статус

- AMG решает миллионные сетки на одной 24 ГБ GPU.
- Кэш и обновление матрицы позволяют выполнять долгие расчёты без перестройки иерархии.
- Подготовлена подробная документация (HTML + PDF).

## Планы развития

1. Оптимизация памяти (смешанная точность, альтернативный коэрснинг).
2. Поддержка JFNK и полностью имплицитных схем.
3. Масштабирование на несколько GPU и внедрение распределённого AMG.
4. Расширение библиотеки физических моделей (полимер, термические эффекты).
