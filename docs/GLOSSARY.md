# Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| **AMG (Algebraic Multigrid)** | Иерархический метод решения линейных систем, строящий грубые сетки по анализу матрицы. |
| **C/F splitting** | Разделение узлов на грубые (C) и тонкие (F) в алгоритме Ruge–Stuben. |
| **IMPES** | Схема «Implicit Pressure, Explicit Saturation»: давление решается имплицитно, насыщенности — явно. |
| **Near-nullspace** | Подпространство «почти нулевых» мод, которые AMG должен воспроизводить (константы, градиенты). |
| **RAP** | Операция построения грубого оператора: $A_c = R A P$, где $R = P^{\mathrm{T}}$. |
| **Smoother** | Итерационный метод («сглаживатель») на каждом уровне AMG, подавляющий высокочастотные ошибки. |
| **TPFA** | Two-Point Flux Approximation — двухточечная схема для потоков в методе конечных объёмов. |
| **Transmissibility** | Коэффициент, связывающий поток между двумя ячейками; зависит от проницаемости и геометрии. |
| **GMRES** | Итерационный метод Крылова для несимметричных систем. |
| **Jacobian-Free Newton–Krylov (JFNK)** | Нелинейный решатель, вычисляющий производные через численное умножение на вектор. |
| **Playwright** | Фреймворк для управления браузером, используется для экспорта PDF. |
| **MkDocs** | Статический генератор документации из Markdown. |
