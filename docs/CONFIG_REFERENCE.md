# Справочник настроек

### CLI ключи (bench/bench.py)
- `--nx --ny --nz`: размеры сетки.
- `--mode`: `fi` | `impes` (если доступен).
- `--steps`: число шагов по времени.
- `--newton`: максимум итераций Ньютона за шаг.
- `--jfnk`: максимум итераций GMRES (fgmres) на итерацию Ньютона.
- `--geo_cycles --geo_pre --geo_post`: параметры V‑цикла AMG (число циклов за вызов, пред/пост‑сглаживания).
- `--geo-tol --geo-max-iter`: критерии остановки AMG на коарс‑уровнях.
- `--log-json-dir`: директория для JSONL‑логов.

### sim_params (основные)
- `p_step_ls_max` (Па): предел шага по давлению в LS (минимум авто).
- `line_search_min_alpha`: минимальный α в LS (по умолчанию 1e-6).
- `trust_radius`: фиксированный trust‑радиус (иначе динамический).
- `cfg_cpr_use_dsdy_hat` (bool): использовать dS/dy в диагонали S‑блока hat.
- `cfg_cpr_phys_sat_cap` (bool): физический кламп насыщенностей в CPR (крупные сетки).
- `debug_diag_f` (bool): печать `[diag F]` изменений между вызовами.

### Переменные окружения
- `OIL_DEBUG=1`: подробные VC/RBGS‑логи.
- `OIL_SIM_SKIP_PATCHES=1`: отключить тестовые «патчи».
- `GEO_RAP_DEBUG_MAX_N`: предел размера для RAP‑проверки (для отладки).
